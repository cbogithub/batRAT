# batRAT

batRAT is a open source remote administration tool coded in C#. It will provide easy-to-use user interface.

### Requirements
- .NET Framework 4.0 Client Profile
- Supported Operating Systems(32- and 64-bit)
- - Windows 7 (Tested)
- - Windows 10 (Tested)
